# 故障排查

## AstrBot API 返回 401

这是未携带 API Key 时的正常结果。请在 AstrBot WebUI 创建仅含 `im` scope 的 Key，并填写到 `config/config.json`。不要把 Key 发到聊天或提交到 Git。

## QQ 命令没有出现

确认 `astrbot_plugin_boss_job_assistant` 已出现在 AstrBot 插件页并已启用；安装后需要重载插件或重启 AstrBot。QQ 会话还必须符合 AstrBot 的白名单/唤醒前缀配置。

## 网页面板显示“连接失败”

先访问 `http://127.0.0.1:17861/health`。若打不开，双击 `启动助手.bat`；若显示 401，检查网页面板中的桥接令牌是否与 `config/config.json` 一致。

## 没收到 QQ 通知

依次确认：AstrBot 在线、API Key 只有 `im` 权限且未撤销、已在目标 QQ 会话执行 `/求职助手 绑定`、本地助手已重启加载新配置。

## 出现验证码或登录失效

助手会停止。请人工完成登录或验证、确认页面恢复正常后，再手动开始任务。本项目不提供验证码绕过或风控规避能力。
